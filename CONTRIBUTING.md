# Contributing to Learn Agentic Patterns

Thanks for your interest in contributing! Here's how to get started.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/learnagenticpatterns.git`
3. Create a branch: `git checkout -b my-feature`
4. Install dependencies: `npm install`
5. Copy `.env.example` to `.env.local` and configure your environment
6. Run the dev server: `npm run dev`

## Making Changes

- Keep PRs focused on a single change
- Follow the existing code style
- Run `npm run lint` before submitting
- Run `npm run build` to verify your changes compile

## What to Contribute

- New agentic patterns or improvements to existing ones
- Bug fixes
- UI/UX improvements
- Documentation improvements
- Accessibility enhancements
- Performance optimizations
- New blog content about agentic AI

## Submitting a Pull Request

1. Push your branch to your fork
2. Open a PR against `main`
3. Describe what your change does and why
4. Link any related issues

## Reporting Issues

Open an issue on GitHub with:
- A clear description of the problem
- Steps to reproduce (if applicable)
- Expected vs actual behavior

## Code of Conduct

Be respectful and constructive. We're all here to learn and build together.
